git workflow

1. git status
 
   " changes in [filename]"
   git diff [filename]
   
   git restore [filename]
 
2. git add [filename] or git add --patch [filename]

   " [filename] or a part of it will be ready for commit"   
   git diff --staged [filename]"

   git restore --staged [filename]
   
3. git commit -m "a descriptive message no longer than 50 chars"


"first pull it than push it"
4. git pull origin [branch-name]

5. git push origin [branch-name]


---------------

how to work with branches

"list branches"
git branch    

"select branch"
git checkout [branch-name]

"you are now in [branch-name]"

--------------

Advance cmds

git add --patch [filename]
git commit --amend --no-edit
git reset --soft HEAD^
git reset --hard HEAD^
